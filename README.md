## MercadoLibre Challenge

**Cliente**
<br /> React
<br /> Css

**Servidor**
<br /> Node
<br /> Express
<br /> Nodemon (watcher)

<br /> <br /> 
**Configuración**
Clonar el repositorio. 
<br /> Ejecutar **yarn setUp**

**Para correr la aplicación**
Ejecutar **yarn dev**
<br /> El cliente debería inicializarse por defecto en localhost:3000 y el server en localhost:5000.
